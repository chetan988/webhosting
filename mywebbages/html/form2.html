Different form input methods we saw in the previous chapter do not pass any information while pressing submit button
We have to associate each input method to a nave and/or value to pass the choosen/input value for backend processing
We will see how we can use different cgi methods to handle the input values

EMAIL TEXT INPUT

syntax : <input type="email" name="login" />
output :

NOTE: while pressing submit see the address bar now we are passing input value associate with the name
ex: file:///home/chetan/study/webhosting/mywebbages/html/form2.html?login=chetan_das%40yahoo.com
NOTE: the '@' sign is converted to '%40' so that predefined html processing will not be confussed

RADIO BUTTON

syntax :
<input type="radio" name="status" value='m'/> married
<input type="radio" name="status" value='u'/> unmarried

output :
married
unmarried

analysis: when clicked married status=m will be passed when clicked unmarried status=u will be passed
ex: file:///home/chetan/study/webhosting/mywebbages/html/form2.html?login=&citizenship=&citizenship_2=0&status=u

CHECKBOX

syntax :
<input type="checkbox" name="pet" value='d'/> dog
<input type="checkbox" name="pet" value='c'/> cat
<input type="checkbox" name="pet" value='f'/> fish

output :
dog
cat
fish

analysis: when check options it will pass name=value pair like pet=d&pet=f if dog and fish are checked
ex: file:///home/chetan/study/webhosting/mywebbages/html/form2.html?login=&citizenship=&citizenship_2=0&pet=d&pet=f



WORKING WITH LINK

tag : <a href=""> some description ... </a>

there are below type link exist

  • Internal : link to pages exist inside same server
  • External : link to pages exist outside server
  • Inpage : Link inside the page itself

INTERNAL LINK

syntax : <a href="helloworld.html"> click me to go hellowrold page </a>
output : click me to go hellowrold page

EXTERNAL LINK

syntax : <a href="google.com"> click me to go google home page </a>
output : click me to go google home page

INPAGE LINK

Inpage links are created to travel in the same page

So we create a inpage tag location where the page will jump while clicked on the link

Then we will create a link on wwhick user will click

syntax for tag : <a name="top" /> : this tag will be created at the top of page where page should jump
syntax for link : <a href="#top"> click me to go top of this page </a>
output : click me to go top of this page
