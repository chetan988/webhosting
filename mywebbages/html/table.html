TABLE IN HTML

We will learn table basics in this chapter

tag: <table> : start of table
<tr> .. </tr> : one row
<th> .. </th> : table head - the content will be bold
<td> .. </td> : table data
syntax :
<h5> MARK SHEET </h5>
<table>
  <tr> <th> SUBJECT </th><th> mark </th> </tr>
  <tr> <td> MATH </td><th> 99 </gtd> </tr>
  <tr> <td> SCIENCE </td><th> 95 <d/gt> </tr>
  <tr> <td> HISTORY </td><th> 78 <d/gt> </tr>
</table>

output :
MARK SHEET
SUBJECTMARK
MATH99
SCIENCE95
HISTORY78
