In previous page we saw we can add style to all paragraph through css. But what if we want different style for one paragraph and another style for another paragraph. Well css provide that option too. We can use class and id to do that.

Now what is a class and id. class is set of css style rules bundle and attached to a class name. A class can be applied many times in one page. Similarly in id we can add multiple css style rule but conventionally id used only once in a page.

lets see some example

class

syntax :     . class name {
     para1:val1;para2:val2;para3:val3;
   }

id

    # id name {
     para1:val1;para2:val2;para3:val3;
   }

Example using both

  <html>
    <head>

    <style type="css/text">
      .large {
       font-size:130%;
       }
      .bold {
       font-weight:bold;
       }
      .underline {
       text-decoration:underline;
      }
      #green {
       color:green;
      }
    </style>
    </head>
    <body>
        <p class="large"> This line contain only one style class </p>
        <p id="green" class="underline"> Css id applied to this line </p>
        <p class="large underline"> This line has two class </p>
        <p> We can use <span class="bold large"> span to </span> add style to certain words </p>
      </body>
    </html>

output :

This line contain only one style class

Css id applied to this line

This line has two class

We can use span to add style to certain words

Note :
** More than one class can be used with space separated list
** ID applied only once on a page
** you can use id and class together
** span tag is used to apply the style in between the paragraph
