CSS : cascade style sheet
css is used to add more visual effect to html

Let us start the learning process

css code can be Embeded in the applicable tags to change their apperance.
This method call inline css. This is not recomended becuse there are lot of cool stuff that can be done.

Lets start with example.

Tag : <p style="parameter1:value1;parameter2:value2;..." > paragraph content </p>
example :
   <p> This line is with out style </p>

   <p> style="color:blue;font-size:150%"> This line is with style </p>
output :   

This line is with out style

  

This line is with style

Disadvantage of inline css

  • In source page both the content and style sheet will be written which will make the code hard to read
  • If we have 500 paragraph so we will have 500 style sheet code to make them look similar
  • In latter stage if we want to make a small change on paragraph we have to do it in all 500 places
To over come above disadvantage we have internal css.
  • The Internal css are usually added in the head tag
  • They are created with tag so when a perticular tag gets added the attached css also get effective
  • Chang the css at one place it will effect thoughout the page
Lets take a simple example :
<html>
  <head>
    <style type="css/text">
      p {
         color:red;
       }
    </style>
  </head>

  <body>
    <p> This is line 1.</p>
    <p> This is line 2.</p>
  </body>
</html>

output :

This is line 1.


This is line 2.



If we will make the color to green it will affect all the paragraph

<html>
  <head>
    <style type="css/text">
      p {
         color:green;
       }
    </style>
  </head>

  <body>
    <p> This is line 1.</p>
    <p> This is line 2.</p>
  </body>
</html>

output :

This is line 1.


This is line 2.
