Floating

The formatting we created always appear one below another. What if we want to put the content in left or right margin? We can do that by float method. Float is a style parameter so added in the style description of the class

Syntax:
float : left | right | clear

clear :both | left | right     #=> clear the floatong effect

Example :

  <html>
    <head>
      <style type="text/css">

         div {
           background:green;
           font-size:130%;
         }

         .redbox {
           background-color:red;
           height:100px;
           float:left;
         }

         .bluebox {
           background-color:blue;
           float:right;
           height:100px;
         }

         .float {
           float:left;
         }

         .floatright {
           float:right;
         }

      </style>
    </head>
    <body>

      < div class="redbox">
         redbox
      </div>

      < div class="bluebox bold">
         bluebox
      </div>

        <p> This line will flow between redbox (on left) and bluebox (on the right)</p>

        <p class="floatleft"> This line will float towards left</p>

        <p class="floatright"> This line will float towards right</p>

        <div class="clear"></div>

       <p> A new paragraph</p>

    </body>
  </html>

Output :

redbox
bluebox


This line will flow between redbox (on left) and bluebox (on the right)

This line will float towards left

This line will float towards right

A new paragraph
